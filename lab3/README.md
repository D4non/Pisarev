# Реализация коллекций и интерфейсов в C#

Этот проект содержит реализации простых коллекций на C# с поддержкой стандартных интерфейсов .NET.

## Реализованные классы

### 1. SimpleList
Класс `SimpleList` реализует простой список с использованием динамического массива.

**Реализуемые интерфейсы:**
- `IEnumerable` - поддержка перебора элементов (foreach)
- `ICollection` - базовые операции: добавление, удаление, подсчёт элементов
- `IList` - индексированный доступ, вставка, удаление по индексу

**Основные методы:**
- `Add(object? value)` - добавляет элемент в конец списка
- `Remove(object? value)` - удаляет первое вхождение элемента
- `RemoveAt(int index)` - удаляет элемент по индексу
- `Insert(int index, object? value)` - вставляет элемент по индексу
- `Clear()` - очищает список
- `Contains(object? value)` - проверяет наличие элемента
- `IndexOf(object? value)` - возвращает индекс элемента
- `CopyTo(Array array, int index)` - копирует элементы в массив

**Свойства:**
- `Count` - количество элементов
- `this[int index]` - доступ к элементу по индексу

### 2. SimpleDictionary<TKey, TValue>
Класс `SimpleDictionary<TKey, TValue>` реализует словарь с использованием хеш-таблицы и цепочек для разрешения коллизий.

**Реализуемые интерфейсы:**
- `IDictionary<TKey, TValue>` - словарь с ключами и значениями
- `IReadOnlyDictionary<TKey, TValue>` - поддержка только чтения

**Основные методы:**
- `Add(TKey key, TValue value)` - добавляет пару ключ-значение
- `Remove(TKey key)` - удаляет элемент по ключу
- `ContainsKey(TKey key)` - проверяет наличие ключа
- `TryGetValue(TKey key, out TValue value)` - пытается получить значение по ключу
- `Clear()` - очищает словарь

**Свойства:**
- `Count` - количество элементов
- `Keys` - коллекция ключей
- `Values` - коллекция значений
- `this[TKey key]` - доступ к значению по ключу

**Особенности реализации:**
- Использует хеш-таблицу с цепочками для разрешения коллизий
- Автоматически увеличивает размер таблицы при достижении коэффициента загрузки 0.75
- Использует простые числа для размера таблицы

### 3. DoublyLinkedList
Класс `DoublyLinkedList` реализует двунаправленный связный список.

**Реализуемые интерфейсы:**
- `IEnumerable` - поддержка перебора элементов
- `ICollection` - базовые операции
- `IList` - индексированный доступ

**Основные методы:**
- `Add(object? value)` - добавляет элемент в конец
- `AddFirst(object? value)` - добавляет элемент в начало
- `AddLast(object? value)` - добавляет элемент в конец
- `Remove(object? value)` - удаляет элемент
- `RemoveAt(int index)` - удаляет элемент по индексу
- `Insert(int index, object? value)` - вставляет элемент по индексу
- `Clear()` - очищает список
- `Contains(object? value)` - проверяет наличие элемента
- `IndexOf(object? value)` - возвращает индекс элемента

**Свойства:**
- `Count` - количество элементов
- `this[int index]` - доступ к элементу по индексу (оптимизирован для доступа с конца)

## Тестирование

Проект включает полный набор автоматизированных тестов, написанных с использованием xUnit.

**Запуск тестов:**
```bash
dotnet test
```

Тесты проверяют:
- Корректность всех операций добавления, удаления, поиска
- Обработку граничных случаев (пустые коллекции, несуществующие элементы)
- Обработку исключений
- Корректность перебора элементов
- Детерминированность операций
