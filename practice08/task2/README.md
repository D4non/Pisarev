## Описание задачи

Задача демонстрирует создание и использование асинхронных итераторов в C# - мощного механизма для работы с асинхронными последовательностями данных. Асинхронные итераторы позволяют обрабатывать данные, которые приходят со временем (например, из сетевых запросов, файловых потоков или баз данных) без блокировки основного потока.

## Используемые технологии и библиотеки
### 1. C# 8.0+

- **Асинхронные потоки (Async Streams)** - ключевая фича C# 8.0

- **```IAsyncEnumerable<T>```** - интерфейс для асинхронных последовательностей

- **```AsyncEnumerator```** - компиляторная магия для асинхронных итераторов

### Пространства имен

- ```System.Collections.Generic``` - содержит IAsyncEnumerable<T>

- ```System.Threading.Tasks``` - для асинхронных операций

### .NET Core 3.0+
- Поддержка асинхронных потоков появилась в этих версиях


## Детали реализации
### Структура программы

```    
static async Task Main()
{
    // Асинхронный перебор элементов
    await foreach (int number in GenerateNumbersAsync(5))
    {
        // Обработка каждого элемента
    }
}
```

```
static async IAsyncEnumerable<int> GenerateNumbersAsync(int count)
{
    // Асинхронная генерация последовательности
    for (int i = 1; i <= count; i++)
    {
        await Task.Delay(200); // Имитация асинхронной работы
        yield return i;        // Возврат элемента последовательности
    }
}
```

### Асинхронный итератор: ```async IAsyncEnumerable<T>```

- ```async``` - указывает, что метод содержит асинхронные операции

- ```IAsyncEnumerable<int>``` - возвращает асинхронную последовательность целых чисел

- Может содержать await выражения и ```yield return```

### ```yield return``` в асинхронном контексте

- Возвращает текущий элемент последовательности

- Сохраняет состояние метода до следующего вызова

- Работает совместно с ```await``` в одном методе (возможность C# 8.0+)

### ```await foreach``` для асинхронного перебора

- Асинхронный аналог ```foreach```

- Ожидает следующий элемент последовательности

- Не блокирует поток во время ожидания
